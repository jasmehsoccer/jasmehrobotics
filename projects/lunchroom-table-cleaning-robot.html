Early Autonomy

Lunchroom Table Cleaning Robot

An early robotics project focused on autonomous cleaning in a familiar school environment.

Demo video

Overview

A compact robot prototype for cleaning lunchroom tables using autonomous movement and path behavior.

Problem

School cleaning tasks are repetitive and time-sensitive. The project asks how a small robot could automate a constrained surface-cleaning workflow.

Architecture

Mobile robot platform with line/path sensing, motor control, and cleaning actuation.

Hardware

Arduino-style controller, motors, chassis, cleaning attachment, and sensor wiring.

Software

Path-following logic, motor commands, and simple autonomous behavior sequencing.

Results

Working early prototype and useful foundation for later robotics projects.

Lessons Learned

Even simple autonomy requires careful mechanical layout, reliable sensing, and repeated testing.

Theme

A first step toward using robots for practical support tasks in everyday spaces.
