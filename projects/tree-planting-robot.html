Sustainability Robotics

Autonomous Tree Planting Robot

A robot built to support reforestation by automating parts of the sapling detection, positioning, and planting workflow.

Project board Open demo video

Overview

An autonomous planting robot that connects mobility, mechanism design, and sensing to an environmental restoration challenge.

Problem

Reforestation is labor-intensive and difficult to scale. The project explores how robotics can assist repetitive outdoor planting tasks.

Architecture

A mobile base coordinates movement, visual/sensor input, and a planting mechanism through a simple autonomous workflow.

Hardware

Omni-directional motion components, motors, servos, robot frame, wiring, and a planting-oriented mechanism.

Software

Control logic for movement, detection, and actuation sequencing.

Results

Received 1st Place in Middle School Engineering and the Lemelson Young Inventors Award.

Lessons Learned

Environmental robotics requires mechanisms that can tolerate messy real-world conditions, not just clean lab tests.

Theme

This project frames robotics as a way to support people working on sustainability and restoration.
